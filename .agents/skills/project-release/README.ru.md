# Project Release Skill

Скилл для релизов ris-claude-code с семантическим версионированием и консистентным workflow.

## Проблема

Процесс релиза подвержен ошибкам: забытые обновления changelog, неправильные версии, пропущенные теги, несогласованные release notes.

## Решение

- **Семантическое версионирование** — MINOR для новых скиллов, PATCH для обновлений
- **Чеклист перед релизом** — проверка всех требований
- **Матрица обновления файлов** — что именно обновлять
- **Верификация после релиза** — подтверждение успеха

## Установка

```bash
cp -r skills/project-release ~/.claude/skills/
```

## Краткий справочник

| Тип изменения | Версия |
|---------------|--------|
| Новый скилл/компонент | MINOR (1.5.0 → 1.6.0) |
| Обновление существующего | PATCH (1.5.0 → 1.5.1) |
| Баг-фикс, документация | PATCH (1.5.0 → 1.5.1) |

## Ключевые фичи

- Чеклист валидации перед релизом
- Правила определения версии
- Матрица решений по файлам
- Пошаговый workflow релиза
- Шаблон GitHub release
- Верификация после релиза
- Гайд по типичным ошибкам
